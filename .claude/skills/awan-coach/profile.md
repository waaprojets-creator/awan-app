## Transitions de phase

### Phase 0 → Phase 1
**Critères requis (TOUS):**
- [ ] ≥ 3 semaines de séances régulières (2–3/sem)
- [ ] RPE perçu < 6 sur les exercices de base à volume MV
- [ ] Aucune douleur articulaire ou tendineuse résiduelle
- [ ] Sommeil moyen 7j ≥ 7h

**Changements Phase 1 :**
- Volume : MEV → MAV progressif (+2 séries/groupe/semaine)
- Intensité : RPE 7–8 (1–2 RIR) autorisé
- Mésocycle : 5 semaines + 1 semaine décharge
- Calories : déficit −250 kcal (recomposition active)

### Phase 1 → Phase 2
**Critères requis :**
- [ ] BF% estimé ≤ 18% (recomposition suffisamment avancée)
- [ ] Volume MAV toléré sans signes de surmenage (récup > 7/10)
- [ ] 1RM retrouvé à ≥ 90% du niveau pré-arrêt (estimation)

**Changements Phase 2 :**
- Périodisation DUP ou bloc (selon préférence)
- Volume MAV → MRV sur groupes prioritaires
- Nutrition selon objectif recalibré (recomp / prise / sèche)
