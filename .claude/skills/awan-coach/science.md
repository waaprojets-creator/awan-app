# Référence scientifique — AWAN Coach

## 1. Formules de base

### BMR — Mifflin-St Jeor (JADA 1990 — plus précise que Harris-Benedict)
```
Homme : BMR = 10×poids(kg) + 6,25×taille(cm) − 5×âge + 5
Femme : BMR = 10×poids(kg) + 6,25×taille(cm) − 5×âge − 161
```

### TDEE — multiplicateurs d'activité
```
1,2   Sédentaire (bureau, <5 000 pas/j)
1,375 Légèrement actif (5 000–10 000 pas/j)
1,55  Modérément actif (10 000 pas/j + 3–4 séances/sem)
1,725 Très actif (travail physique + 5+ séances/sem)
1,9   Extrêmement actif (athlète professionnel)
```
*Note : TDEE = BMR × multiplicateur — méthode la plus utilisée en pratique clinique.*

### 1RM — Formule Epley (NSCA recommandée)
```
1RM = poids × (1 + reps / 30)
```
*Source : NSCA Essentials of Strength and Conditioning, 4e édition.*

### Estimation 1RM par RPE (Helms 2017)
```
RIR 4 = ~89% 1RM
RIR 3 = ~92% 1RM
RIR 2 = ~95% 1RM
RIR 1 = ~97% 1RM
RIR 0 = 100% 1RM (échec)
```

## 2. Protéines

### Recommandations générales (JISSN Position Stand 2017)
```
Minimum absolu       : 1,4 g/kg/j
Optimum hypertrophie : 1,6–2,2 g/kg/j
Déficit calorique    : 2,2–2,4 g/kg/j (préservation LBM prioritaire)
Retourneur           : 2,0 g/kg/j dès J1 (activation muscle memory)
```

### Distribution des repas (Areta 2013 — J Physiol)
```
Optimal : 4–5 repas de 30–40 g protéines espacés de 3–4h
Seuil leucine par repas : 0,03 g/kg → 2,5 g leucine pour 82 kg
Au-delà de 40 g/repas : bénéfice marginal supplémentaire faible
```

### Timing (Aragon & Schoenfeld 2013, 2017)
```
Fenêtre anabolique : 2h post-entraînement (pas critique si distribution totale OK)
Pré-entraînement   : protéines + glucides 1–2h avant
Post-entraînement  : 30–40 g protéines dans les 2h
Avant sommeil      : caséine ou protéine lente (Res 2012 — +22% MPS nocturne)
```

## 3. Macronutriments complets

### Lipides
```
Minimum santé hormonale : 0,5 g/kg/j (testostérone dépendante des lipides)
Recommandé musculation  : 0,7–1,0 g/kg/j
Source : ISSN, Hamalainen 1984 (testostérone + lipides alimentaires)
```

### Glucides
```
Calcul : (calories totales − protéines×4 − lipides×9) / 4
Timing : priorité autour de l'entraînement (pré + post)
Indice glycémique : non critique si apport total correct
```

### Fibres
```
Recommandation EFSA : 25–38 g/j
Bénéfices : satiété, microbiome, glycémie stable
```

### Hydratation
```
Base          : 35–40 ml/kg/j = 2,9–3,3 L/j pour 82 kg
Entraînement  : +500–750 ml/h
Déshydratation > 2% poids → −10 à −20% performance (NSCA)
```

## 4. Volume d'entraînement — Landmarks RP (Israetel)

### Définitions
```
MV  (Maintenance Volume)       : volume minimum pour maintenir la masse existante
MEV (Minimum Effective Volume) : premier seuil de stimulus hypertrophique
MAV (Maximum Adaptive Volume)  : zone optimale de croissance (MEV → MRV)
MRV (Maximum Recoverable Vol.) : plafond — au-delà, la récupération échoue
```

### Table par groupe musculaire — intermédiaire naturel
| Groupe | MV | MEV | MAV | MRV | Source |
|---|---|---|---|---|---|
| Pectoraux | 6 | 8 | 12–20 | 22+ | RP Hypertrophy 2020 |
| Dos | 8 | 10 | 14–22 | 25+ | RP Hypertrophy 2020 |
| Épaules | 6 | 8 | 16–22 | 26+ | RP Hypertrophy 2020 |
| Biceps | 4 | 6 | 14–20 | 26+ | RP Hypertrophy 2020 |
| Triceps | 4 | 6 | 10–18 | 22+ | RP Hypertrophy 2020 |
| Quadriceps | 6 | 8 | 12–18 | 20+ | RP Hypertrophy 2020 |
| Ischio-jambiers | 4 | 6 | 10–16 | 20+ | RP Hypertrophy 2020 |
| Fessiers | 0 | 0 | 4–12 | 16+ | RP Hypertrophy 2020 |
| Mollets | 6 | 8 | 12–16 | 20+ | RP Hypertrophy 2020 |
| Abdos | 0 | 0 | 16–20 | 25+ | RP Hypertrophy 2020 |

*Note Phase 0 : utiliser 60–70% des valeurs MV ci-dessus.*

### Progression intra-mésocycle
```
+1 à 2 séries/groupe/semaine jusqu'à atteindre MAV
Arrêt si : douleur, récupération < 6/10, performance stagnante 2 sem consécutives
```

## 5. Intensité — RPE / RIR

### Principes (Refalo 2024 — PMC11127506)
```
Zone hypertrophie maximale : 0–3 RIR (ne pas dépasser 4 RIR pour les sets de travail)
Echec musculaire           : occasionnel, dernière série uniquement
Phase 0 cible              : 3–4 RIR (adaptation sans surmenage)
Phase 1 cible              : 1–2 RIR (stimulus optimal)
Phase 2 cible              : 0–2 RIR selon exercice
```

### Table RPE ↔ % 1RM approximatif
```
RPE 6  = ~3–4 RIR = ~82–86% 1RM
RPE 7  = ~3 RIR   = ~88% 1RM
RPE 8  = ~2 RIR   = ~92% 1RM
RPE 9  = ~1 RIR   = ~96% 1RM
RPE 10 = ~0 RIR   = 100% 1RM (échec)
```

## 6. Fréquence d'entraînement

### Méta-analyses clés
```
Schoenfeld 2016 (PubMed 27102172) :
  2× par semaine > 1× par semaine pour l'hypertrophie

Schoenfeld 2024 (PubMed 41343037) — méta-régression 67 études :
  Volume est le facteur dominant
  Fréquence : effet marginal si volume équivalent
  Recommandation : 2×/sem par groupe = rapport qualité/récupération optimal
```

### Application Phase 0
```
3 séances full body / semaine
Groupes musculaires sollicités 3× (stimulus fréquent, volume faible)
Repos minimum : 1 jour entre séances
```

## 7. Périodisation

### Comparaison LP vs DUP (Frontiers Public Health 2026)
```
29 études analysées :
LP (Linéaire)  : efficace sur force et hypertrophie à long terme
DUP (ondulant) : légèrement supérieur sur la force, équivalent hypertrophie
Conclusion     : les deux fonctionnent — la cohérence > le modèle
```

### Structure mésocycle recommandée (Israetel 2020)
```
Durée        : 5 semaines progressives + 1 semaine décharge
Progression  : +1–2 séries/groupe/sem · RIR descend de 4 à 1
Décharge     : volume −50% · intensité inchangée · 5–7 jours
Fréquence    : décharge toutes les 4–6 semaines (Coleman & Israetel 2024)
```

### Protocole Phase 0 spécifique (retourneur 1 an)
```
Semaines 1–2 : MV (6 séries/groupe) · RPE 6 (4 RIR) · full body 2×/sem
Semaines 3–4 : MV+1 série · RPE 7 (3 RIR) · full body 3×/sem
Critère sortie Phase 0 : cf. profile.md § Transitions
```

## 8. Sommeil & récupération

### Impact quantifié (Lamon 2021 — PMC7785053)
```
< 6h/nuit :
  MPS (synthèse protéique musculaire) : −18%
  Cortisol plasmatique                : +21%
  Testostérone plasmatique            : −24%
  GH (hormone de croissance)          : significativement réduite

< 5h/nuit sur 5 jours :
  Myofibrillar protein synthesis      : −18–19%
```

### Recommandations
```
Cible        : 7–9h (OMS adulte 18–64 ans)
Critique     : < 7h = règle sleep.short_avg devrait se déclencher
Application  : jours post-entraînement prioritaires (pic GH nocturne J+1)
```

### Score récupération subjectif (proxy HRV — Halson 2014)
```
1–3  : surmenage → réduire volume −30%, pas d'échec musculaire
4–6  : attention → maintenir volume, pas d'augmentation
7–8  : normal → progression standard
9–10 : optimal → légère surcharge autorisée
```

## 9. Anthropométrie & composition corporelle

### Formule Navy (BF% sans appareil)
```
Homme : BF% = 86,01 × log₁₀(taille_cm − cou_cm) − 70,041 × log₁₀(hauteur_cm) + 36,76
Précision : ±3% (suffisant pour suivi de tendance)
Mesures requises : tour de taille (nombril) + tour de cou + hauteur
```

### FFMI (Fat-Free Mass Index — Kouri 1995)
```
FFMI = LBM(kg) / hauteur(m)²
Plafond naturel prouvé  : 25,0
Athlètes élites naturels : 24–25
Intermédiaire            : 20–22
Valeur actuelle profil   : 19,7
```

### Jalons visuels BF% homme
```
25%+ : ventre proéminent, peu de tonus visible
23%  : (profil actuel) mou, peu de définition
20%  : légère vascularisation avant-bras
17%  : abdos visibles en contraction
15%  : abdos toujours visibles, bonne définition
12%  : définition complète, veines marquées
```

### Ratio santé cardiovasculaire
```
WHR  = tour taille / tour hanches   → cible homme < 0,90
WHtR = tour taille / hauteur (cm)   → cible < 0,50 (meilleur prédicteur mortalité cv)
```

### Fréquence de mesure recommandée
```
Poids          : quotidien, matin à jeun → utiliser moyenne 7j (Israetel 2019)
Tour de taille : hebdomadaire, matin
Autres circonf.: mensuel (bras, cuisse, mollet, poitrine)
Tour de cou    : mensuel (requis formule Navy)
BF% estimé     : mensuel (calcul Navy depuis mensurations)
```

## 10. Suppléments — hiérarchie evidence-based

### Tier 1 — Effet prouvé, risque nul
```
Créatine monohydrate : 3–5 g/j (maintenance) · pas de phase de charge nécessaire
                       Source : JISSN Position Stand 2017 (PMC5469049)
                       Effet  : +8–14% force · +1–2 kg masse maigre à 12 sem

Protéines en poudre  : si apport alimentaire < 1,6 g/kg/j uniquement
```

### Tier 2 — Bénéfice modéré, contexte-dépendant
```
Caféine      : 3–6 mg/kg, 45–60 min avant · performance aiguë +2–4%
Vitamine D   : si < 30 ng/mL sérique (déficit fréquent, lié à testostérone)
Oméga-3      : 2–3 g EPA+DHA/j · anti-inflammatoire · MPS +Smith 2011
Magnésium    : 400 mg/j bisglycinate · qualité sommeil si déficit
Bêta-alanine : 3,2–6,4 g/j · tampon acide lactique · utile >10 reps
```

### Tier 3 — Preuves insuffisantes ou marginales
```
Tout le reste : BCAA (inutile si protéines suffisantes), HMB, glutamine,
                boosters pré-workout complexes, boosters testostérone
```

## 11. Muscle memory — science de la reprise

### Mécanisme (Gundersen 2016 — Exp Gerontol)
```
Les myonoyaux acquis lors de l'entraînement persistent dans les fibres
musculaires pendant des mois à des années après l'arrêt complet.
→ Lors de la reprise : pas de novo myogenèse mais RÉACTIVATION
→ Temps de regain : 2–4× plus rapide qu'un vrai débutant
```

### Implications pratiques (profil : 1 an sans entraînement)
```
Semaines 1–4 : gains rapides = adaptation neurologique (pas encore hypertrophie)
Semaines 5–12 : muscle memory s'active → gains visibles · regain masse accéléré
Mois 3–6     : retour au niveau pré-arrêt potentiel si nutrition + sommeil optimaux
```

### Erreur à éviter absolument
```
Commencer à volume MAV ou MRV → surmenage garanti → blessure potentielle
Le tendon et le tissu conjonctif récupèrent plus lentement que le muscle
Règle : muscle memory ≠ permission de sauter la Phase 0
```
